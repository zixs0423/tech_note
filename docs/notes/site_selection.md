# Site Selection

## WKT

* WKT (Well-Known Text): a text-based markup language used to represent vector geometry objects on a map or geographical coordinate system, standardized by the Open Geospatial Consortium (OGC).
  * POINT: A single XY coordinate. e.g. POINT (30 10)
  * POLYGON: A closed shape. Note the double parentheses; the first and last point must match to close the loop. e.g. POLYGON ((30 10, 40 40, 20 40, 10 20, 30 10))
